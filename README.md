⚙️#TCC_CTI_CPPSE#

Landing page desenvolvida para apresentar cabines de pintura para peças de pequeno porte com sistema de exaustão.


📌 Sobre

O projeto tem como objetivo demonstrar, de forma clara e objetiva, o funcionamento e os benefícios das cabines de pintura, utilizando uma interface simples e informativa.


🚀 Tecnologias

HTML
CSS


🖥️ Funcionalidades

Apresentação do projeto
Explicação do sistema de exaustão

▶️ Como executar
git clone https://github.com/seu-usuario/TCC_CTI_CPPSE.git
cd TCC_CTI_CPPSE

Abra o arquivo index.html no navegador.

📄 Observação


Projeto desenvolvido para fins acadêmicos (TCC).
